# LENS Production - Professional Videography Website

A stunning, high-performance website for LENS Production, a professional videography and photography company based in Tunisia.

## 🌟 Features

- **Modern Design**: Dark theme with cinematic aesthetics and smooth animations
- **3D Background**: Interactive Three.js background with subtle animations
- **Responsive**: Fully responsive design for all devices
- **SEO Optimized**: Comprehensive SEO with structured data, sitemap, and meta tags
- **Performance**: Optimized for Core Web Vitals with lazy loading and image optimization
- **PWA Ready**: Progressive Web App support with manifest
- **Social Media Integration**: Links to Instagram (50k+) and Facebook (110k+)

## 🚀 Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **3D Graphics**: Three.js, React Three Fiber
- **Deployment**: Vercel

## 📦 Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## 🌐 Deployment

### Vercel (Recommended)

1. Push your code to GitHub
2. Import project in Vercel
3. Vercel will auto-detect Next.js and deploy
4. Configure custom domain if needed

The `vercel.json` is already configured for optimal deployment.

### Environment Variables

No environment variables required for basic deployment.
